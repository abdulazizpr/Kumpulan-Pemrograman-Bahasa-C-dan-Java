(gr13s2) Graf 13 Shift 2


Pembuat Soal: Rosa A. S.

Batas Waktu Eksekusi 5 Detik
Batas Memori 1 MB

Kompetensi yang dievaluasi: Kemampuan untuk mengoperasikan graf. Diberikan sebuah graf. Hitung banyaknya elemen jalur dan elemen simpul graf setelah dihapus. Wajib menggunakan penelusuran jalur graf dari simpul awal sampai akhir.

Format Masukan:

n, 0 < n < 50, banyaknya jalur.
n baris jalur dengan format kota_awal kota_akhir
kota yang dihapus

Format Keluaran:

jumlah elemen simpul dan elemen jalur yang tersisa

Contoh Masukan

5
A B
A C
B C
C D
D C
B

Contoh Keluaran

6
